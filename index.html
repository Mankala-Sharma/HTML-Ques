Difference between Class and Id

Class can be same for different elements but Id should be unique for every element.

As on above example first Div and first span have same class but second div and second span having unique id.

Difference between block and inline

Block elements always occupies the width of screen and dispaly on next line.

Inline elements occupies the content width and dispaly on same line.

Difference between em and strong

strong tag is used to provide strong importance and em tag is used to give emphasis.
